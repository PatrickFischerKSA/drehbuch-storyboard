Drehbuch ↓ Storyboard MP4

Konvertiert jedes Drehbuch automatisch in ein animiertes Storyboard-Video. Inklusive KI-Prompts für Midjourney, Stable Diffusion und DALL·E. Komplett lokal. Keine API nötig.

🐍 Python 3.10+ 🎬 MoviePy 🖼 Pillow ✅ Keine API 📄 MIT Lizenz
↓ GitHub Repository → Anleitung
Was es kann

Vollautomatisch.
Null Konfiguration.

01
Drehbuch parsen

Erkennt automatisch INT./EXT.-Szenenkopf, Regieanweisungen, Figurennamen und Dialoge im Standard-Screenwriting-Format.

02
Image Prompts

Generiert für jede Szene einen optimierten englischen Prompt — direkt verwendbar in Midjourney, Stable Diffusion oder DALL·E.

03
Storyboard-Karten

Erstellt visuelle 1280×720 Karten mit Szenennummer, Beschreibung, Dialog und Prompt. Mit Platzhalter für das KI-Bild.

04
MP4-Export

Alle Szenen werden zu einem abspielbaren MP4-Video zusammengefügt. Einstellbare Anzeigedauer pro Szene.

Workflow

In 4 Schritten
zum Film.

①
Drehbuch schreiben
Normales .txt-File im Standard-Format
②
Programm ausführen
Ein Befehl im Terminal
③
Prompts nutzen
Bilder mit KI-Tools generieren
④
Film schneiden
Bilder ersetzen — fertig!
Installation & Verwendung

Drei Befehle.
Läuft sofort.

TERMINAL — bash
# 1. Repository klonen
$ git clone https://github.com/patrickmfischer/drehbuch-storyboard.git
$ cd drehbuch-storyboard
 
# 2. Abhängigkeiten installieren
$ pip install -r requirements.txt
 
# 3a. Demo starten (Mephisto-Beispiel)
$ python3 drehbuch_zu_storyboard.py --demo
 
# 3b. Eigenes Drehbuch verarbeiten
$ python3 drehbuch_zu_storyboard.py mein_film.txt
 
# Mit Optionen (10s pro Szene, eigener Dateiname)
$ python3 drehbuch_zu_storyboard.py mein_film.txt -d 10 -o film.mp4
 
✅ MP4 gespeichert: film.mp4
✨ Fertig! 6 Szenen · ~60 Sekunden
